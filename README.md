# Dinesh & Nithiya Engagement Invite

A React + Vite engagement invitation website.

## Local Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Render

This repo includes `render.yaml` for one-click setup.

If you configure manually in Render dashboard:

- Runtime: `Static Site`
- Build Command: `npm install && npm run build`
- Publish Directory: `dist`

After linking this repo in Render, every push to the default branch auto-deploys.
